================================================================================
NEW PLACEHOLDERS IDENTIFIED
================================================================================

${AccountDesignation} - User's role/designation (e.g., "Registrar", "Admin")
                      - Source: roles.role_name from users table

${DateGenerated} - Date and time when report was generated
                  - Format: YYYY-MM-DD HH:MM:SS (e.g., "2024-01-15 14:30:00")
                  - Source: Current timestamp when report is generated

${PageNo} - Current page number
            - Format: Integer (e.g., 1, 2, 3)
            - Source: FPDF's PageNo() method

${TotalPages} - Total number of pages in the document
                - Format: Integer (e.g., 5)
                - Source: FPDF's internal page counter (via reflection)
                - Example usage: "Page ${PageNo} of ${TotalPages}" displays as "Page 1 of 5"

${YearLevel} - Student's year level (for Student reports only)
              - Source: students.year_level
              - Format: "1st Year", "2nd Year", "3rd Year", "4th Year"

${Section} - Student's section (for Student reports only)
            - Source: students.section
            - Format: As stored in database (e.g., "1/1-1", "2/1-1")

================================================================================
FULL NAME FORMATTING LOGIC
================================================================================

STUDENT & FACULTY FULL NAME:
Format: "${LastName}, ${FirstName} ${MiddleName}"

Examples:
- If MiddleName exists: "Dela Cruz, Juan Dela"
- If MiddleName is NULL: "Cruz, Juan"
- If MiddleName is empty: "Cruz, Juan"

Note: MiddleName should be displayed on a second line if space allows,
      or appended after FirstName if single line is preferred.

Current interpretation shows MiddleName on a separate line for better readability.
