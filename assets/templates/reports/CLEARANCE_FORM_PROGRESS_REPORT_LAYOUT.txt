================================================================================
SPACING & SIZING GUIDELINES
================================================================================

HEADER SECTION:
- Logo: ~30mm width, positioned at top-left (X: 15mm, Y: 10mm)
- Report Title: Font size 14-16pt, Bold, Centered
- School Year/Semester: Font size 10-12pt, Regular
- Department/Generated By: Font size 9-10pt, Regular
- Spacing between header elements: ~5-7mm

SUMMARY SECTION:
- Section title: Font size 12pt, Bold
- Label + Value pairs: Font size 10pt, Regular
- Spacing between summary items: ~5mm
- Margin from header: ~10mm

TABLE SECTION:
- Table header: Font size 9-10pt, Bold
- Table data: Font size 8-9pt, Regular
- Row height: ~6-8mm
- Column padding: ~2-3mm
- Border thickness: ~0.5mm
- Margin from summary: ~10mm

FOOTER:
- Page numbers: Font size 8pt, Italic, Centered
- Position: Bottom of page (Y: ~285mm on A4)

================================================================================
COLOR SCHEME SUGGESTIONS
================================================================================

Header Background: Light gray or white with border
Table Header Background: Light blue/gray (#E8F4F8 or #F0F0F0)
Table Borders: Gray (#CCCCCC)
Text Color: Black (#000000)
Logo: Full color or grayscale

================================================================================
NOTES FOR TEMPLATE DESIGN
================================================================================

1. Header placeholders should be replaced with actual values
2. Summary values are numbers (integers)
3. Table rows repeat for each student/faculty record
4. If a field is NULL (like MiddleName), display as blank or "-"
5. Status values are formatted: "Unapplied", "In Progress", "Complete"
6. Employment Status only appears in Faculty reports
7. For students, "Department" shows Program Name, not Department Name
8. Tables should support pagination if there are many rows
